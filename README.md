## Description:
This repository contains the source code, libraries, and assets for an Arduino-based sonner project that incorporates a self-powered LCD display.

## Components:
1. Arduino R3
2. sg90 Servo motor
3. LCD6102 Display
4. Ultrasonic Distance Sensor
5. Potentiometer(10KÎ©)
